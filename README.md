# JobForYou — Рекрутинговое агентство премиум-класса

Веб-приложение для рекрутингового агентства, позволяющее кандидатам оставлять анкеты и работодателям искать сотрудников.

## Особенности

- 👤 **Для кандидатов**: поиск вакансий, создание резюме, подача анкеты
- 🏢 **Для работодателей**: поиск сотрудников, размещение вакансий
- 📱 **Отзывчивый дизайн**: работает на мобильных, планшетах и ПК
- 🔒 **Безопасность**: секреты хранятся на сервере, не видны в коде
- ⚡ **Быстрая отправка**: интеграция с Telegram для уведомлений

## Технологии

- **Frontend**: HTML5, CSS3, JavaScript (ванильный)
- **Backend**: Node.js, Express.js
- **API**: Telegram Bot API
- **Среда**: dotenv для управления переменными окружения

## Установка и запуск

### Требования
- Node.js (версия 14+)
- npm
- Токен бота Telegram

### Шаги

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/YOUR_USERNAME/job-for-you.git
cd job-for-you
```

2. **Установите зависимости:**
```bash
npm install
```

3. **Создайте файл `.env`:**
```bash
cp .env.example .env
```

4. **Добавьте ваши секреты в `.env`:**
```
TOKEN=YOUR_TELEGRAM_BOT_TOKEN
CHAT_ID=YOUR_CHAT_ID
PORT=3000
```

Как получить токен и chat_id:
- **Токен**: откройте Telegram, найдите `@BotFather`, создайте бота и скопируйте токен
- **Chat ID**: откройте бота, отправьте сообщение, перейдите на `https://api.telegram.org/bot{TOKEN}/getUpdates` и найдите `chat.id`

5. **Запустите сервер:**
```bash
npm start
```

6. **Откройте в браузере:**
- Для просмотра всех страниц: `http://localhost:3000`
- Форма кандидата: `http://localhost:3000/resume.html`

## API Endpoints

### POST `/api/send-telegram`

Отправляет анкету кандидата в Telegram.

**Тело запроса:**
```json
{
  "name": "Иван Петров",
  "phone": "+41 77 123 4567",
  "position": "Водитель",
  "message": "Опыт вождения 5 лет"
}
```

**Ответ (успех):**
```json
{
  "ok": true,
  "data": { ... }
}
```

**Ответ (ошибка):**
```json
{
  "error": "Сообщение об ошибке"
}
```

## Безопасность

⚠️ **Важно:**
- **Никогда** не коммитьте `.env` файл в Git
- Хранить токены только на сервере
- Используйте `.gitignore` для исключения секретов
- Если токен был случайно опубликован, сразу измените его в Telegram

## Live Server (локальная разработка)

Если хотите использовать Live Server в VS Code:

1. Установите расширение "Live Server"
2. Откройте `resume.html` и нажмите "Go Live"
3. Сервер на `http://localhost:3000` должен быть запущен одновременно
4. Форма будет отправлять запросы на бэкенд через CORS

## Развёртывание

Для развёртывания на хостинге (Heroku, Railway, Vercel и т.д.):

1. Добавьте в `.env` переменные окружения хоста
2. Убедитесь, что `package.json` содержит правильный `"start"` скрипт
3. Загрузите на платформу и следуйте их инструкциям

Пример для Heroku:
```bash
heroku create your-app-name
git push heroku main
heroku config:set TOKEN="your_token"
heroku config:set CHAT_ID="your_chat_id"
```
